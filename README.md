## English

**ErogeRichPresence** is a lightweight Python script designed to implement eroge game activity in **Discord Rich Presence**.  
It supports both Japanese and English.

### Supported Titles
- Dohna Dohna: Let's Do Bad Things Together  
- Hentai Prison
